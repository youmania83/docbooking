## 🔧 How to Customize (3 Easy Steps)

### Step 1: Change Doctor Data
**File**: `lib/data.ts`

```typescript
// Modify existing doctor or add new one
{
  id: "5",
  name: "Dr. Your Name",
  specialty: "Your Specialty",
  fee: 400,
  slots: ["10:00 AM", "2:00 PM", "4:00 PM"],
}
```

### Step 2: Change Colors
**In**: Any `.tsx` file

Search & replace:
- `bg-blue-600` → `bg-green-600` (button color)
- `text-blue-600` → `text-green-600` (text color)

### Step 3: Change Text
**File**: `app/page.tsx` (for homepage)

Edit the headline, subtext, button labels directly in the JSX.
